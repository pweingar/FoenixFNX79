# FoenixKBD79

Firmware for the Foenix FNXKBD79 PS/2 keyboard.

## Special Commands

* Set RGB -- Set the "on" color of an RGB LED
* Turn LED or Off -- Set whether an RGB LED is on or off
* Refresh LEDs -- Update the LEDs

## Special Key Scancodes

* "MON" --
* "APP LIST" --
* "MENU HELP" --
* Left "blank" --
* Right "blank" --
